HTML Cheatsheet
  1. In this Section we are going to learn about the basics of HTML.

    HTML Boilerplate

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <!-- Body -->
</body>
</html>

  2. Headings

    There are six headings available in HTML, H1 is the largest among all, and H6 is the smallest. 
    <h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>


  3. Container

    Container tags are the tags that contain some data such as text, image, etc. There are several container tags in HTML.

    div tag

    The div tag or division tag is used to make blocks or divisions in the document. 
    <div>This is div block </div>

    span tag

    The span is a container for inline content 
    <span>This is span block </span>

    pre tag

    The pre tag represents pre-formatted text 
    <pre>Hello World </pre>

    Paragraph tag

    The p tag is used to create a paragraph in HTML 
    <p>This is a paragraph</p>


  4. Text Formatting

    Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable.

    bold tag

    <b>I'm bold text</b>

    italic tag

    <i>I'm italic text</i>

    subscript tag

    <sub>I'm subscript text</sub>

    superscript tag

    <sup>I'm superscript text</sup>

  5. Lists

    Lists can be either numerical, alphabetic, bullet, or other symbols. You can specify list type and list items in HTML for the clean document.

    ul tag

    <ul>
<li>Your Data</li>
<li>Your Data</li>
</ul>

    ol tag

    <ol>
<li>Data 1</li>
<li>Data 2</li>
<li>Data 3</li>
</ol>

  6. Media

    Media is anything that is present in digital form such as image, video, audio, etc.

    audio tag

    It is used to embed sound content in the document. 
    <audio> controls>
<source src="demo.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

    img tag

    It is used to embed or import image in a webpage. 
    <img src="Source_of_image" alt="Alternate text">

    video tag

    It is used to embed video in the webpage. 
    <video> width="480" height="320" controls>
<source src="demo_move.mp4" type="video/mp4">
Your browser does not support the video tag.
</video>
